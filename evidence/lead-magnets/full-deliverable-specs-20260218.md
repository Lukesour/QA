# Lead Magnet Full Deliverable Specs (2026-02-18)

Use this as the deliverable standard replacing skeleton-only status.

## General Standard

- Every asset must include:
  - Usage scenario (who/when).
  - Input fields and required data.
  - Output example.
  - Failure/stop-loss notes.
  - DM handoff sentence.
- Delivery SLA: send within 5 minutes after keyword trigger.

## LM001-LM009

- LM001 三笔账评估表:
  - Inputs: tuition, living cost, visa/IHS, opportunity cost, target salary.
  - Output: conservative/aggressive payback months + stop-loss threshold.
- LM002 一年制适配度自测表:
  - Inputs: goal clarity, execution hours/week, budget buffer, job path confidence.
  - Output: fit score + recommended path (apply now / delay / alternative route).
- LM003 选校四维矩阵:
  - Inputs: course fit, employability, cost, admission feasibility.
  - Output: chase/target/safe list and replacement rationale.
- LM004 留学预算规划器:
  - Inputs: fixed and variable monthly costs, exchange-rate stress test.
  - Output: 12-month cashflow and risk buffer warning.
- LM005 文书结构模板:
  - Inputs: experiences, quantified outcomes, target module/career fit.
  - Output: 4-paragraph evidence-driven structure with weak-point flags.
- LM006 低GPA补强清单:
  - Inputs: GPA gap, available timeline, current portfolio status.
  - Output: 90-day weekly actions ranked by ROI.
- LM007 求职时间线模板:
  - Inputs: intake month, target role, CV readiness.
  - Output: pre-arrival + post-arrival 24-week timeline.
- LM008 Offer对比决策表:
  - Inputs: curriculum, city cost, visa/work pathway, fallback option.
  - Output: weighted score and backup-switch trigger.
- LM009 咨询前诊断问卷:
  - Inputs: country, major, budget, intake, background, language, target role, stage.
  - Output: auto-routing to content handoff vs diagnostic call.
